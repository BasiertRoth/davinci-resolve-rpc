# davinci-resolve-rpc
Discord Rich Presence for DaVinci Resolve - shows your current page, project name and time. Free &amp; Studio supported.

I looked everywhere for a working Discord RPC tool for DaVinci Resolve, couldn't find anything that actually worked, so I built my own.
Shows your DaVinci Resolve activity as a Discord status in real time, including the current page (Edit, Color, Fusion, Fairlight, Cut, Deliver, Media Pool), your project name, and how long you've been in the app.


Disclaimer: AI tools were used as a supplementary aid for debugging.
